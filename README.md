# Batch Simulate Protocols
This set of scripts is meant to help simulate protocols to ensure they are ready for customer delivery. When this applicated is initiated, it will also the user to choose a folder of protocols or a single protocol. The user can then add an optional folder containing any custom labware needed within any of the protocols. The protocol will create many different versions of each protocol to randomize the run time parameters. After all protcol are generated, they are all simulate using opentrons_simulate. Failed protocols will be captured and have their problematic parameters put into a report. Each simulation can be found in the simulation_raw_outputs folder for manual investigation
